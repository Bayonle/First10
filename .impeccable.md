# Impeccable Design Context — First10

## Design Context

### Users
FRSC Ogun Command duty officers working the dispatch desk — 24/7 shifts. Their job:
scan a live queue, spot the real high-severity incident in seconds, act
(dispatch/arrive/transport), and close the loop. Time-to-action target ≤15s per
ticket (paper §7.1). Secondary surface: the dev-only local-chat cockpit used by the
engineering team to simulate bystanders — functional, not precious.

### Brand Personality
Operational command center. Urgent but never noisy: urgency is carried by severity
color and queue pressure numbers, not decoration. The console should feel like
mission equipment — a room where the map glows and the queue is worked.

### Aesthetic Direction
**"Incident command"** (adopted 23 Jul 2026, superseding the "dispatch manifest"
paper aesthetic): dark-only near-black canvas (cool hue, never pure #000), raised
charcoal cards with 1px hairlines, rounded-xl geometry, pill-shaped status chips.
Three-pane layout: queue (400px) · live corridor map · detail panel. Blue is the
action color (Dispatch); red belongs to high severity and hard alarms only; amber
= medium/warnings; green = live/positive. Severity also paints the map markers and
the 1px card edge-bars. Mono type for ids, coordinates, clocks, KPI numbers.

Fonts: **Archivo** (headings), **Public Sans** (body/UI), **Spline Sans Mono**
(timestamps, coordinates, IDs, KPI figures).

Anti-references: neon cyberpunk glow, glassmorphism, gradient text, light-theme
admin templates.

### Design Principles
1. Severity is the loudest thing on screen — red is spent nowhere else.
2. Evidence vs. inference stays visually distinct (reporter messages ≠ system/AI
   annotations) — D-013 is a design rule, not just a data rule.
3. The map is situational awareness, the queue is the work order — selection stays
   synchronized between them.
4. Every state is legible at a glance: pills for status, filled/hollow dots for
   progress, KPI numbers in mono.
5. Motion only for state changes (new ticket arriving, map fly-to on selection) —
   subtle, fast, honest.
